  Install RNA-BRIQ using the repository https://github.com/QBio-ai/QM3D-RNA.git
* Step 2:
  Add the directory RNAdataset and add the files in https://github.com/marcellszi/rna3db/releases/download/2025-10-01-incremental-release/rna3db-mmcifs.tar.xz

  Run rnadb.py to convert mmcif files in PDB and use RNA-BRIQ to generate files with dot-bracket secondary structure annotations
